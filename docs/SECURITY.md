# Security Policy

## Supported Versions

Only the latest published version on npm receives security patches.

## Reporting a Vulnerability

Please **do not** open a public issue for security vulnerabilities.

Report privately via [GitHub Security Advisories](https://github.com/mkemeter/obsidian-vaultgate-mcp/security/advisories/new)
or email the maintainer directly.

Expect an acknowledgement within 48 hours and a patch within 14 days for confirmed vulnerabilities.
